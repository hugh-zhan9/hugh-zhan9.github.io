PrinterWriter 简介

PrintWriter是字符类型的打印输出流，它继承于Writer。

PrintWriter用于向文本输出流打印对象的格式化表示形式。它实现了PrintStream中所有的print方法。它不包含用于写入原始字节的方法，对于这些字节，程序应该使用未编码的字节流进行写入。
