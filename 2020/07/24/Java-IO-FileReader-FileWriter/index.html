FileReader 和 FileWriter 简介

FileReader是用于读取字符流的类，它继承于InputStreamReader。要读取原始字节流，考虑使用 FileInputStream。

FileWriter是用于写入字符流的类，它继承于OutputStreamWriter。要写入原始字节流，考虑使用 FileOutputStream。
