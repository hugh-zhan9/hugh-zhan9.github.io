FileInputStream 和 FileOutputStream 简介

FileInputStream是文件输入流，它继承于InputStream。
通常，使用FileInputStream从某个文件中获得输入字节。

FileOutputStream是文件输出流，它继承于OutputStream。
通常，使用FileOutputStream将数据写入File或 FileDescriptor的输出流。
