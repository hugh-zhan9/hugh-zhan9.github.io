File 简介

File是文件和目录路径名的抽象表示形式。

File直接继承于Object，实现了Serializable接口和Comparable接口，意味着File对象支持序列化操作。而实现Comparable接口，意味着File对象之间可以比较大小；File能直接被存储在有序集合(如TreeSet、TreeMap)中。
