HashSet 介绍

HashSet 简介

HashSet是一个没有重复元素的集合。由HashMap实现的，不保证元素的顺序，而且HashSet允许使用null元素。

HashSet是非同步的。如果多个线程同时访问一个HashSet，而其中至少一个线程修改了该set，那么它必须保持外部同步。这通常是通过对自然封装该set的对象执行同步操作来完成的。如果不存在这样的对象，则应该使用Collections.synchronizedSet方法来“包装”set。最好在创建时完成这一操作，以防止对该set进行意外的不同步访问：
