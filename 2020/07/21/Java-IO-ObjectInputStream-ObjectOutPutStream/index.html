ObjectInputStream 和 ObjectOutputStream 简介

ObjectOutputStream和ObjectInputStream地作用是：对基本数据和对象进行序列化操作支持。

创建文件输出流对应的ObjectOutputStream对象，该ObjectOutputStream对象能提供对基本数据或对象的持久存储。当我们需要读取这些存储的基本数据或对象时，可以创建文件输入流对应的ObjectInputStream，进而读取出这些基本数据或对象。

注意：只有支持java.io.Serializable或java.io.Externalizable接口的对象才能被ObjectInputStream/ObjectOutputStream所操作!
