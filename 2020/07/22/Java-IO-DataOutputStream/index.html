DataOutputSream 简介

DataOutputStream是数据输出流。它继承于FilterOutputStream。

DataOutputStream是用来装饰其它输出流，将DataOutputStream和DataInputStream输入流配合使用，允许应用程序以与机器无关方式从底层输入流中读写基本 Java数据类型。
