图形界面

发表于 2020-04-05 更新于 2020-05-08 分类于 Python 热度： Valine：
本文字数： 4.3k 阅读时长 ≈ 7 分钟

Python支持多种图形界面的第三方库，包括

  • Tk
  • wxWidgets
  • Qt
  • GTK

等等，但是Python自带的库式支持Tk的Tkinter，使用Tkinter，无需安装任何包就可以直接使用。因此这里就简单地使用Tkinter进行GUI编程。

Tkinter

我们来梳理一下概念：

  • 我们编写地Python代码会调用内置地Tkinter，Tkinter封装了访问Tk地接口；
  • Tk是一个图形库，支持多种操作系统，使用Tcl语言开发
  • Tk会调用操作系统提供地本地GUI接口，完成最终地GUI

所以我们的代码只需要调用Tkinter提供的接口就可以了。

海龟绘图（Turtle Graphics）后来被移植到各种高级语言中，Python内置了turtle库，基本上100%复制了原始的Turtle Graphics的所有功能。
