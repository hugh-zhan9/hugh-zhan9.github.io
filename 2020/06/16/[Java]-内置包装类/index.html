еҢ…иЈ…зұ»гҖҒиЈ…з®ұе’ҢжӢҶз®ұ

иҷҪз„¶ Java еҸҜд»ҘзӣҙжҺҘеӨ„зҗҶеҹәжң¬зұ»еһӢпјҢдҪҶжҳҜеңЁжңүдәӣжғ…еҶөдёӢйңҖиҰҒе°Ҷе…¶дҪңдёәеҜ№иұЎжқҘеӨ„зҗҶпјҢиҝҷж—¶е°ұйңҖиҰҒе°Ҷе…¶иҪ¬жҚўдёәеҢ…иЈ…зұ»гҖӮ

йӮЈд№ҲеҰӮдҪ•жҠҠдёҖдёӘеҹәжң¬зұ»еһӢи§ҶдёәеҜ№иұЎ(еј•з”Ёзұ»еһӢ)пјҹжҜ”еҰӮжҲ‘д»¬жғіиҰҒжҠҠintеҹәжң¬зұ»еһӢеҸҳжҲҗдёҖдёӘеј•з”Ёзұ»еһӢпјҢеҸҜд»Ҙе®ҡд№үдёҖдёӘIntegerзұ»пјҢе®ғеҸӘеҢ…еҗ«дёҖдёӘе®һдҫӢеӯ—ж®өintпјҢиҝҷж ·Integerзұ»е°ұеҸҜд»Ҙи§Ҷдёәintзҡ„еҢ…иЈ…зұ»(Wrapper Class):

иЈ…з®ұе’ҢжӢҶз®ұ

еҹәжң¬ж•°жҚ®зұ»еһӢиҪ¬жҚўдёәеҢ…иЈ…зұ»зҡ„иҝҮзЁӢз§°дёәиЈ…з®ұпјҢдҫӢеҰӮжҠҠintеҢ…иЈ…жҲҗIntegerзұ»зҡ„еҜ№иұЎпјӣеҢ…иЈ…зұ»еҸҳдёәеҹәжң¬ж•°жҚ®зұ»еһӢзҡ„иҝҮзЁӢз§°дёәжӢҶз®ұпјҢдҫӢеҰӮжҠҠIntegerзұ»зҡ„еҜ№иұЎз®ҖеҢ–жҲҗintгҖӮ

жүӢеҠЁе®һдҫӢеҢ–дёҖдёӘеҢ…иЈ…зұ»з§°дёәжүӢеҠЁжӢҶз®ұиЈ…з®ұгҖӮJava 1.5 зүҲжң¬д№ӢеүҚеҝ…йЎ»жүӢеҠЁжӢҶз®ұиЈ…з®ұпјҢд№ӢеҗҺеҸҜд»ҘиҮӘеҠЁжӢҶз®ұиЈ…з®ұпјҢд№ҹе°ұжҳҜеңЁиҝӣиЎҢеҹәжң¬ж•°жҚ®зұ»еһӢе’ҢеҜ№еә”зҡ„еҢ…иЈ…зұ»иҪ¬жҚўж—¶пјҢзі»з»ҹе°ҶиҮӘеҠЁиҝӣиЎҢиЈ…з®ұеҸҠжӢҶз®ұж“ҚдҪңпјҢдёҚз”ЁеңЁиҝӣиЎҢжүӢе·Ҙж“ҚдҪңпјҢдёәејҖеҸ‘иҖ…жҸҗдҫӣдәҶжӣҙеӨҡзҡ„ж–№дҫҝгҖӮдҫӢеҰӮпјҡ

еҢ…иЈ…зұ»зҡ„еә”з”Ё

  1. е®һзҺ°intе’ҢIntegerзҡ„зӣёдә’иҪ¬жҚў

    еҸҜд»ҘйҖҡиҝҮIntegerзұ»зҡ„жһ„йҖ ж–№жі•е°ҶintиЈ…з®ұпјҢйҖҡиҝҮIntegerзұ»зҡ„intValueе°ҶIntegerжӢҶз®ұгҖӮ

  2. е°Ҷеӯ—з¬ҰдёІиҪ¬жҚўдёәж•°еҖјзұ»еһӢ

    • Integerзұ»

    • Floatзұ»

  3. е°Ҷж•ҙж•°иҪ¬жҚўдҪҚеӯ—з¬ҰдёІ

    Integerзұ»жңүдёҖдёӘйқҷжҖҒзҡ„toString()ж–№жі•пјҢеҸҜд»Ҙе°Ҷж•ҙж•°иҪ¬жҚўдёәеӯ—з¬ҰдёІгҖӮ

дёҚеҸҳзұ»

жүҖжңүзҡ„еҢ…иЈ…зұ»еһӢйғҪжҳҜдёҚеҸҳзұ»пјҢжҹҘзңӢIntegerзҡ„жәҗз ҒпјҢж ёеҝғд»Јз ҒеҰӮдёӢпјҡ

еӣ жӯӨдёҖж—ҰеҲӣе»әдәҶIntegerеҜ№иұЎпјҢиҜҘеҜ№иұЎе°ұжҳҜдёҚеҸҳзҡ„гҖӮеҜ№дёӨдёӘIntegerе®һдҫӢжҜ”иҫғеӣ дёәIntegerжҳҜеј•з”Ёзұ»еһӢжүҖд»ҘдҪҝз”Ёequals()гҖӮ

еҸҜд»ҘеҸ‘зҺ°пјҢ==жҜ”иҫғпјҢиҫғе°Ҹзҡ„дёӨдёӘзӣёеҗҢзҡ„Integerиҝ”еӣһtrueпјҢиҫғеӨ§зҡ„дёӨдёӘзӣёеҗҢзҡ„Integerиҝ”еӣһfalseпјҢиҝҷжҳҜеӣ дёәIntegerжҳҜдёҚеҸҳзұ»пјҢзј–иҜ‘еҷЁжҠҠInteger x = 127;иҮӘеҠЁеҸҳдёәInteger x = Integer.valueOf(127);пјҢдёәдәҶиҠӮзңҒеҶ…еӯҳпјҢInteger.valueOf()еҜ№дәҺиҫғе°Ҹзҡ„ж•°пјҢе§Ӣз»Ҳиҝ”еӣһзӣёеҗҢзҡ„е®һдҫӢпјҢеӣ жӯӨпјҢ==жҜ”иҫғвҖңжҒ°еҘҪвҖқдёәtrueпјҢдҪҶжҲ‘д»¬з»қдёҚиғҪеӣ дёәJavaж ҮеҮҶеә“зҡ„IntegerеҶ…йғЁжңүзј“еӯҳдјҳеҢ–е°ұз”Ё==жҜ”иҫғпјҢеҝ…йЎ»з”Ёequals()ж–№жі•жҜ”иҫғдёӨдёӘIntegerгҖӮ

еӣ дёәInteger.valueOf()еҸҜиғҪе§Ӣз»Ҳиҝ”еӣһеҗҢдёҖдёӘIntegerе®һдҫӢпјҢеӣ жӯӨпјҢеңЁжҲ‘д»¬иҮӘе·ұеҲӣе»әIntegerзҡ„ж—¶еҖҷпјҢд»ҘдёӢдёӨз§Қж–№жі•пјҡ

  • ж–№жі•1пјҡInteger n = new Integer(100);
  • ж–№жі•2пјҡInteger n = Integer.valueOf(100);

ж–№жі•2жӣҙеҘҪпјҢеӣ дёәж–№жі•1жҖ»жҳҜеҲӣе»әж–°зҡ„Integerе®һдҫӢпјҢж–№жі•2жҠҠеҶ…йғЁдјҳеҢ–з•ҷз»ҷIntegerзҡ„е®һзҺ°иҖ…еҺ»еҒҡпјҢеҚідҪҝеңЁеҪ“еүҚзүҲжң¬жІЎжңүдјҳеҢ–пјҢд№ҹжңүеҸҜиғҪеңЁдёӢдёҖдёӘзүҲжң¬иҝӣиЎҢдјҳеҢ–гҖӮ

жҲ‘д»¬жҠҠиғҪеҲӣе»әвҖңж–°вҖқеҜ№иұЎзҡ„йқҷжҖҒж–№жі•з§°дёәйқҷжҖҒе·ҘеҺӮж–№жі•гҖӮInteger.valueOf()е°ұжҳҜйқҷжҖҒе·ҘеҺӮж–№жі•пјҢе®ғе°ҪеҸҜиғҪең°иҝ”еӣһзј“еӯҳзҡ„е®һдҫӢд»ҘиҠӮзңҒеҶ…еӯҳгҖӮйқҷжҖҒе·ҘеҺӮж–№жі•

еҲӣе»әж–°еҜ№иұЎж—¶пјҢдјҳе…ҲйҖүз”ЁйқҷжҖҒе·ҘеҺӮж–№жі•иҖҢдёҚжҳҜnewж“ҚдҪңз¬ҰгҖӮ

еҰӮжһңжҲ‘д»¬иҖғеҜҹByte.valueOf()ж–№жі•зҡ„жәҗз ҒпјҢеҸҜд»ҘзңӢеҲ°пјҢж ҮеҮҶеә“иҝ”еӣһзҡ„Byteе®һдҫӢе…ЁйғЁжҳҜзј“еӯҳе®һдҫӢпјҢдҪҶи°ғз”ЁиҖ…е№¶дёҚе…іеҝғйқҷжҖҒе·ҘеҺӮж–№жі•д»ҘдҪ•з§Қж–№ејҸеҲӣе»әж–°е®һдҫӢиҝҳжҳҜзӣҙжҺҘиҝ”еӣһзј“еӯҳзҡ„е®һдҫӢгҖӮ

Objectзұ»

ObjectжҳҜJavaзұ»еә“дёӯзҡ„дёҖдёӘзү№ж®Ҡзұ»пјҢд№ҹжҳҜжүҖжңүзұ»зҡ„зҲ¶зұ»гҖӮд№ҹе°ұжҳҜиҜҙпјҡJavaе…Ғи®ёжҠҠд»»дҪ•зұ»еһӢзҡ„еҜ№иұЎиөӢеҖјз»ҷObjectзұ»еһӢзҡ„еҸҳйҮҸгҖӮ

ж–№жі• иҜҙжҳҺ
clone() еҲӣе»әдёҺиҜҘеҜ№иұЎзҡ„зұ»зӣёеҗҢзҡ„ж–°еҜ№иұЎгҖӮ
equals(object) жҜ”иҫғдёӨеҜ№иұЎжҳҜеҗҰзӣёзӯүгҖӮ
finalize() еҪ“еһғеңҫеӣһж”¶еҷЁзЎ®е®ҡдёҚеӯҳеңЁеҜ№иҜҘеҜ№иұЎзҡ„жӣҙеӨҡеј•з”Ёж—¶пјҢеҜ№иұЎеһғеңҫеӣһж”¶еҷЁи°ғз”ЁиҜҘж–№жі•гҖӮ
getClass() иҝ”еӣһдёҖдёӘеҜ№иұЎиҝҗиЎҢж—¶зҡ„е®һдҫӢзұ»гҖӮ
hashCode() иҝ”еӣһиҜҘеҜ№иұЎзҡ„ж•ЈеҲ—з ҒеҖјгҖӮ
notify() жҝҖжҙ»зӯүеҫ…еңЁиҜҘеҜ№иұЎзҡ„зӣ‘и§ҶеҷЁдёҠзҡ„дёҖдёӘзәҝзЁӢгҖӮ
notiflyAll() жҝҖжҙ»зӯүеҫ…еңЁиҜҘеҜ№иұЎзҡ„зӣ‘и§ҶеҷЁдёҠзҡ„е…ЁйғЁзәҝзЁӢгҖӮ
toString() иҝ”еӣһиҜҘеҜ№иұЎзҡ„еӯ—з¬ҰдёІиЎЁзӨәгҖӮ
wait() еңЁе…¶д»–зәҝзЁӢи°ғз”ЁжӯӨеҜ№иұЎзҡ„notify()ж–№жі•жҲ–notifyAll()ж–№жі•еүҚпјҢеҜјиҮҙеҪ“еүҚзәҝзЁӢзӯүеҫ…гҖӮ

toString()ж–№жі•

toString()ж–№жі•иҝ”еӣһиҜҘеҜ№иұЎзҡ„еӯ—з¬ҰдёІпјҢеҪ“зЁӢеәҸиҫ“еҮәдёҖдёӘеҜ№иұЎжҲ–иҖ…жҠҠжҹҗдёӘеҜ№иұЎе’Ңеӯ—з¬ҰдёІиҝӣиЎҢиҝһжҺҘиҝҗз®—ж—¶пјҢзі»з»ҹдјҡиҮӘеҠЁи°ғз”ЁиҜҘеҜ№иұЎзҡ„toString()ж–№жі•иҝ”еӣһиҜҘеҜ№иұЎзҡ„еӯ—з¬ҰдёІиЎЁзӨәгҖӮObject зұ»зҡ„ toString()ж–№жі•иҝ”еӣһвҖңиҝҗиЎҢж—¶зұ»еҗҚ@еҚҒе…ӯиҝӣеҲ¶е“ҲеёҢз ҒвҖқж јејҸзҡ„еӯ—з¬ҰдёІпјҢдҪҶеҫҲеӨҡзұ»йғҪйҮҚеҶҷдәҶ Object зұ»зҡ„toString()ж–№жі•пјҢз”ЁдәҺиҝ”еӣһеҸҜд»ҘиЎЁиҝ°иҜҘеҜ№иұЎдҝЎжҒҜзҡ„еӯ—з¬ҰдёІгҖӮ

е“ҲеёҢз Ғ(hashCode)пјҢжҜҸдёӘJavaеҜ№иұЎйғҪз”Ёе“ҲеёҢз ҒеұһжҖ§пјҢе“ҲеёҢз ҒеҸҜд»Ҙз”ЁжқҘж ҮиҜҶеҜ№иұЎпјҢд»ҘжҸҗй«ҳеҜ№иұЎеңЁйӣҶеҗҲж“ҚдҪңдёӯзҡ„жү§иЎҢж•ҲзҺҮгҖӮ

equals()ж–№жі•

==иҝҗз®—з¬ҰжҳҜжҜ”иҫғдёӨдёӘеј•з”ЁеҸҳйҮҸжҳҜеҗҰжҢҮеҗ‘еҗҢдёҖдёӘе®һдҫӢпјҢequals()ж–№жі•жҳҜжҜ”иҫғдёӨдёӘеҜ№иұЎзҡ„еҶ…е®№жҳҜеҗҰзӣёзӯүпјҢйҖҡеёёеӯ—з¬ҰдёІзҡ„жҜ”иҫғеҸӘжҳҜе…іеҝғеҶ…е®№жҳҜеҗҰзӣёзӯүгҖӮ

getClass()ж–№жі•

getClass()ж–№жі•иҝ”еӣһеҜ№иұЎжүҖеұһзҡ„зұ»пјҢжҳҜдёҖдёӘClassеҜ№иұЎгҖӮйҖҡиҝҮClassеҜ№иұЎеҸҜд»ҘиҺ·еҸ–иҜҘзұ»зҡ„еҗ„з§ҚдҝЎжҒҜпјҢеҢ…жӢ¬зұ»еҗҚгҖҒзҲ¶зұ»д»ҘеҸҠе®ғжүҖе®һзҺ°зҡ„жҺҘеҸЈеҗҚз§°гҖӮ

  • getName()пјҡиҺ·еҸ–еҗҚз§°
  • getSuperclass()пјҡиҺ·еҸ–зҲ¶зұ»
  • getInterfaces()пјҡиҺ·еҸ–е®һзҺ°зҡ„жҺҘеҸЈ

жҺҘеҸ—д»»ж„Ҹеј•з”Ёзұ»еһӢзҡ„еҜ№иұЎ

ж—ўз„¶Objectзұ»жҳҜжүҖжңүеҜ№иұЎзҡ„зҲ¶зұ»пјҢеҲҷжүҖжңүзҡ„еҜ№иұЎйғҪеҸҜд»Ҙеҗ‘ObjectиҝӣиЎҢиҪ¬жҚўпјҢеңЁиҝҷе…¶дёӯд№ҹеҢ…еҗ«дәҶж•°з»„е’ҢжҺҘеҸЈзұ»еһӢпјҢеҚідёҖеҲҮзҡ„еј•з”Ёж•°жҚ®зұ»еһӢйғҪеҸҜд»ҘдҪҝз”ЁObjectиҝӣиЎҢжҺҘж”¶гҖӮ

иҷҪз„¶жҺҘеҸЈдёҚиғҪз»§жүҝдёҖдёӘзұ»пјҢдҪҶжҳҜдҫқз„¶жҳҜObject зұ»зҡ„еӯҗзұ»пјҢеӣ дёәжҺҘеҸЈжң¬иә«жҳҜеј•з”Ёж•°жҚ®зұ»еһӢпјҢжүҖд»ҘеҸҜд»ҘиҝӣиЎҢеҗ‘дёҠиҪ¬еһӢж“ҚдҪңгҖӮеҗҢзҗҶпјҢд№ҹеҸҜд»ҘдҪҝз”Ё ObjectжҺҘж”¶дёҖдёӘж•°з»„пјҢеӣ дёәж•°з»„жң¬иә«д№ҹжҳҜеј•з”Ёж•°жҚ®зұ»еһӢгҖӮ

Integerзұ»

Integerзұ»еңЁеҜ№иұЎдёӯеҢ…иЈ…дәҶдёҖдёӘеҹәжң¬зұ»еһӢintзҡ„еҖјпјҢIntegerзұ»еҜ№иұЎеҢ…еҗ«дёҖдёӘintзұ»еһӢзҡ„еӯ—ж®өгҖӮжӯӨеӨ–пјҢиҜҘзұ»жҸҗдҫӣдәҶеӨҡдёӘж–№жі•пјҢиғҪеңЁintзұ»еһӢе’ҢStringзұ»еһӢд№Ӣй—ҙдә’зӣёиҪ¬жҚўпјҢиҝҳжҸҗдҫӣдәҶеӨ„зҗҶintзұ»еһӢж—¶зҡ„дёҖдәӣеёёйҮҸе’Ңж–№жі•гҖӮ

Integerзұ»зҡ„жһ„йҖ ж–№жі•

Integerзұ»дёӯзҡ„жһ„йҖ ж–№жі•жңүд»ҘдёӢдёӨдёӘпјҡ

  • Integer(int)пјҡжһ„йҖ дёҖдёӘж–°еҲҶй…Қзҡ„IntegerеҜ№иұЎпјҢе®ғиЎЁзӨәжҢҮе®ҡзҡ„intеҖјгҖӮ
  • Integer(string)пјҡжһ„йҖ дёҖдёӘж–°еҲҶй…Қзҡ„IntegerеҜ№иұЎпјҢе®ғиЎЁзӨәStringеҸӮж•°жүҖжҢҮзӨәзҡ„intеҖјгҖӮ

Integerзұ»зҡ„еёёз”Ёж–№жі•

ж–№жі• иҝ”еӣһеҖј еҠҹиғҪ
byteValue() byte д»Ҙbyteзұ»еһӢиҝ”еӣһиҜҘIntegerзҡ„еҖј
shortValue() short д»Ҙshortзұ»еһӢиҝ”еӣһиҜҘIntegerзҡ„еҖј
intValue() int д»Ҙintзұ»еһӢиҝ”еӣһиҜҘIntegerзҡ„еҖј
toString() String иҝ”еӣһдёҖдёӘиЎЁзӨәиҜҘIntegerеҖјзҡ„StringеҜ№иұЎ
equals(Object) boolean жҜ”иҫғжӯӨеҜ№иұЎдёҺжҢҮе®ҡеҜ№иұЎжҳҜеҗҰзӣёзӯү
compareTo(anotherInteger) int еңЁж•°еӯ—дёҠжҜ”иҫғдёӨдёӘInteger еҜ№иұЎпјҢеҰӮзӣёзӯүпјҢеҲҷиҝ”еӣһ 0пјӣ
еҰӮи°ғз”ЁеҜ№иұЎзҡ„ж•°еҖје°ҸдәҺ anotherlntegerзҡ„ж•°еҖјпјҢеҲҷиҝ”еӣһиҙҹеҖјпјӣ
еҰӮи°ғз”ЁеҜ№иұЎзҡ„ж•°еҖјеӨ§дәҺ anotherlntegerзҡ„ж•°еҖјпјҢеҲҷиҝ”еӣһжӯЈеҖј
valueOf(string) Integer иҝ”еӣһдҝқеӯҳжҢҮе®ҡзҡ„StringеҖјзҡ„IntegerеҜ№иұЎ
parseInt(string) int е°Ҷж•°еӯ—еӯ—з¬ҰдёІиҪ¬жҚўдёәintж•°еҖј

Integerзұ»зҡ„еёёйҮҸ

Integerзұ»еҢ…еҗ«д»ҘдёӢ4дёӘеёёйҮҸпјҡ

  • MAX_VALUEпјҡеҖјдёә2^31-1зҡ„еёёйҮҸпјҢе®ғиЎЁзӨә int зұ»еһӢиғҪеӨҹиЎЁзӨәзҡ„жңҖеӨ§еҖјгҖӮ
  • MIN_VALUEпјҡеҖјдёә`-2^31зҡ„еёёйҮҸпјҢе®ғиЎЁзӨә int зұ»еһӢиғҪеӨҹиЎЁзӨәзҡ„жңҖе°ҸеҖјгҖӮ
  • SIZEпјҡз”ЁдәҢиҝӣеҲ¶иЎҘз ҒеҪўејҸиЎЁзӨәintеҖјзҡ„жҜ”зү№дҪҚж•°гҖӮ
  • TYPEпјҡиЎЁзӨәеҹәжң¬зұ»еһӢintзҡ„Classе®һдҫӢгҖӮ

Floatзұ»

Floatзұ»дёӯзҡ„жһ„йҖ ж–№жі•жңүд»ҘдёӢ3дёӘпјҡ

  • Float(double value)пјҡжһ„йҖ дёҖдёӘж–°еҲҶй…Қзҡ„FloatеҜ№иұЎпјҢе®ғиЎЁзӨәиҪ¬жҚўдёәfloatзұ»еһӢзҡ„еҸӮж•°гҖӮ
  • Float(float value)пјҡжһ„йҖ дёҖдёӘж–°еҲҶй…Қзҡ„FloatеҜ№иұЎпјҢе®ғиЎЁзӨәеҹәжң¬зҡ„floatеҸӮж•°гҖӮ
  • Float(String s)пјҡжһ„йҖ дёҖдёӘж–°еҲҶй…Қзҡ„FloatеҜ№иұЎпјҢе®ғиЎЁзӨәStringеҸӮж•°жүҖжҢҮзӨәзҡ„floatеҖјгҖӮ

еңЁFloatзұ»еҶ…йғЁеҢ…еҗ«дәҶдёҖдәӣе’Ңfloatж“ҚдҪңжңүе…ізҡ„ж–№жі•пјҡ

ж–№жі• иҝ”еӣһеҖј еҠҹиғҪ
byteValue() byte д»Ҙbyteзұ»еһӢиҝ”еӣһиҜҘFloatзҡ„еҖјгҖӮ
doubleValue() double д»Ҙdoubleзұ»еһӢиҝ”еӣһиҜҘFloatзҡ„еҖјгҖӮ
floatValue() float д»Ҙfloatзұ»еһӢиҝ”еӣһиҜҘFloatзҡ„еҖјгҖӮ
intValue() int д»Ҙintзұ»еһӢиҝ”еӣһиҜҘFloatзҡ„еҖјпјҲејәеҲ¶иҪ¬жҚўдёәintзұ»еһӢпјүгҖӮ
longValue() long д»Ҙlongзұ»еһӢиҝ”еӣһиҜҘFloatзҡ„еҖјпјҲејәеҲ¶иҪ¬жҚўдёәlongзұ»еһӢпјүгҖӮ
shortValue() short д»Ҙshortзұ»еһӢиҝ”еӣһиҜҘFloatзҡ„еҖјпјҲејәеҲ¶иҪ¬жҚўдёәshortзұ»еһӢпјүгҖӮ
isNaN() boolean еҰӮжһңжӯӨFloatеҖјжҳҜдёҖдёӘйқһж•°еӯ—еҖјпјҢеҲҷиҝ”еӣһtrueпјҢеҗҰеҲҷиҝ”еӣһfalseгҖӮ
isNaN(float v) boolean еҰӮжһңжҢҮе®ҡзҡ„еҸӮж•°жҳҜдёҖдёӘйқһж•°еӯ—еҖјпјҢеҲҷиҝ”еӣһtrueпјҢеҗҰеҲҷиҝ”еӣһfalseгҖӮ
toString() String иҝ”еӣһдёҖдёӘиЎЁзӨәиҜҘFloatеҖјзҡ„StringеҜ№иұЎгҖӮ
valueOf(string s) Float иҝ”еӣһдҝқеӯҳжҢҮе®ҡзҡ„StringеҖјзҡ„FloatеҜ№иұЎгҖӮ
parseFloat(String s) float е°Ҷж•°еӯ—еӯ—з¬ҰдёІиҪ¬жҚўдёәfloatж•°еҖјгҖӮ

Floatзұ»зҡ„еёёз”ЁеёёйҮҸ

еңЁFloatзұ»дёӯеҢ…еҗ«дәҶи®ёеӨҡеёёйҮҸпјҢе…¶дёӯиҫғдёәеёёз”Ёзҡ„еёёйҮҸеҰӮдёӢпјҡ

  • MAX_VALUEпјҡеҖјдёә1.4E38зҡ„еёёйҮҸпјҢе®ғиЎЁзӨәfloatзұ»еһӢиғҪеӨҹиЎЁзӨәзҡ„жңҖеӨ§еҖјгҖӮ
  • MIN_VALUEпјҡеҖјдёә3.4E-45зҡ„еёёйҮҸпјҢе®ғиЎЁзӨәfloatзұ»еһӢиғҪеӨҹиЎЁзӨәзҡ„жңҖе°ҸеҖјгҖӮ
  • MAX_EXPONENTпјҡжңүйҷҗfloatеҸҳйҮҸеҸҜиғҪе…·жңүзҡ„жңҖеӨ§жҢҮж•°гҖӮ
  • MIN_EXPONENTпјҡжңүйҷҗfloatеҸҳйҮҸеҸҜиғҪе…·жңүзҡ„жңҖе°ҸжҢҮж•°гҖӮ
  • MIN_NORMALпјҡдҝқеӯҳfloatзұ»еһӢж•°еҖјзҡ„жңҖе°Ҹж ҮеҮҶеҖјзҡ„еёёйҮҸпјҢеҚі2^-126гҖӮ
  • NaNпјҡдҝқеӯҳfloatзұ»еһӢзҡ„йқһж•°еӯ—еҖјзҡ„еёёйҮҸгҖӮ
  • SIZEпјҡз”ЁжқҘд»ҘдәҢиҝӣеҲ¶иЎҘз ҒеҪўејҸиЎЁзӨәfloatеҖјзҡ„жҜ”зү№дҪҚж•°гҖӮ
  • TYPEпјҡиЎЁзӨәеҹәжң¬зұ»еһӢfloatзҡ„Classе®һдҫӢгҖӮ

дёӢйқўзҡ„д»Јз Ғжј”зӨәдәҶFloatзұ»дёӯеёёйҮҸзҡ„дҪҝз”Ёпјҡ

Doubleзұ»

Doubleзұ»еңЁеҜ№иұЎдёӯеҢ…иЈ…дәҶдёҖдёӘеҹәжң¬зұ»еһӢdoubleзҡ„еҖјгҖӮDoubleзұ»еҜ№иұЎеҢ…еҗ«дёҖдёӘdoubleзұ»еһӢзҡ„еӯ—ж®өгҖӮжӯӨеӨ–пјҢиҜҘзұ»иҝҳжҸҗдҫӣдәҶеӨҡдёӘж–№жі•еҸҜд»Ҙе°Ҷdoubleзұ»еһӢдёҺStringзұ»еһӢзӣёдә’иҪ¬жҚўпјҢеҗҢж—¶иҝҳжҸҗдҫӣдәҶеӨ„зҗҶdoubleзұ»еһӢж—¶жҜ”иҫғеёёз”Ёзҡ„еёёйҮҸе’Ңж–№жі•гҖӮ

doubleзұ»дёӯзҡ„жһ„йҖ ж–№жі•

Doubleзұ»дёӯзҡ„жһ„йҖ ж–№жі•жңүеҰӮдёӢдёӨдёӘпјҡ

  • Double(double value)пјҡжһ„йҖ дёҖдёӘж–°еҲҶй…Қзҡ„DoubleеҜ№иұЎпјҢе®ғиЎЁзӨәиҪ¬жҚўдёәdoubleзұ»еһӢзҡ„еҸӮж•°гҖӮ
  • Double(String s)пјҡжһ„йҖ дёҖдёӘж–°еҲҶй…Қзҡ„DoubleеҜ№иұЎпјҢе®ғиЎЁзӨәStringеҸӮж•°жүҖжҢҮзӨәзҡ„doubleеҖјгҖӮ

doubleзұ»зҡ„еёёз”Ёж–№жі•

ж–№жі• иҝ”еӣһеҖј еҠҹиғҪ
byteValue() byte д»Ҙbyteзұ»еһӢиҝ”еӣһиҜҘDoubleзҡ„еҖј
doubleValue() double д»Ҙdoubleзұ»еһӢиҝ”еӣһиҜҘDoubleзҡ„еҖј
floatValue() float д»Ҙfloatзұ»еһӢиҝ”еӣһиҜҘDoubleзҡ„еҖј
intValue() int д»Ҙintзұ»еһӢиҝ”еӣһиҜҘDoubleзҡ„еҖјпјҲејәеҲ¶иҪ¬жҚўдёәintзұ»еһӢпјү
longValue() long д»Ҙlongзұ»еһӢиҝ”еӣһиҜҘDoubleзҡ„еҖјпјҲејәеҲ¶иҪ¬жҚўдёәlongзұ»еһӢпјү
shortValue() short д»Ҙshortзұ»еһӢиҝ”еӣһиҜҘDoubleзҡ„еҖјпјҲејәеҲ¶иҪ¬жҚўдёәshortзұ»еһӢпјү
isNaN() boolean еҰӮжһңжӯӨDoubleеҖјжҳҜдёҖдёӘйқһж•°еӯ—еҖјпјҢеҲҷиҝ”еӣһtrueпјҢеҗҰеҲҷиҝ”еӣһfalse
isNaN(double v) boolean еҰӮжһңжҢҮе®ҡзҡ„еҸӮж•°ж—¶дёҖдёӘйқһж•°еӯ—еҖјпјҢеҲҷиҝ”еӣһtrueпјҢеҗҰеҲҷиҝ”еӣһfalse
toString() String иҝ”еӣһдёҖдёӘиЎЁзӨәиҜҘDoubleеҖјзҡ„StringеҜ№иұЎ
valueOf(String s) Double иҝ”еӣһдҝқеӯҳжҢҮе®ҡзҡ„StringеҖјзҡ„DoubleеҜ№иұЎ
parseDouble(String s) double е°Ҷж•°еӯ—еӯ—з¬ҰдёІиҪ¬жҚўдёәDoubleж•°еҖј

Doubleзұ»зҡ„еёёз”ЁеҸҳйҮҸ

еңЁDoubleзұ»дёӯеҢ…еҗ«дәҶеҫҲеӨҡеёёйҮҸпјҢе…¶дёӯиҫғдёәеёёз”Ёзҡ„еёёйҮҸеҰӮдёӢпјҡ

  • MAX_VALUEпјҡеҖјдёә1.8E308зҡ„еёёйҮҸпјҢе®ғиЎЁзӨәdoubleзұ»еһӢзҡ„жңҖеӨ§жӯЈжңүйҷҗеҖјзҡ„еёёйҮҸгҖӮ
  • MIN_VALUEпјҡеҖјдёә4.9E-324зҡ„еёёйҮҸпјҢе®ғиЎЁзӨәdoubleзұ»еһӢзҡ„жңҖе°ҸжӯЈйқһйӣ¶еҖјзҡ„еёёйҮҸгҖӮ
  • NaNпјҡдҝқеӯҳdoubleзұ»еһӢзҡ„йқһж•°еӯ—еҖјзҡ„еёёйҮҸ
  • NEGATIVE_INFINITYпјҡдҝқжҢҒdoubleзұ»еһӢзҡ„иҙҹж— з©·еӨ§зҡ„еёёйҮҸ
  • POSITIVE_INFINITYпјҡдҝқжҢҒdoubleзұ»еһӢзҡ„жӯЈж— з©·еӨ§зҡ„еёёйҮҸ
  • SIZEпјҡз”ЁжқҘд»ҘдәҢиҝӣеҲ¶иЎҘз ҒеҪўејҸиЎЁзӨәdoubleеҖјзҡ„жҜ”зү№дҪҚж•°
  • TYPEпјҡиЎЁзӨәеҹәжң¬зұ»еһӢdoubleзҡ„Classе®һдҫӢ

Numberзұ»

NumberжҳҜдёҖдёӘжҠҪиұЎзұ»пјҢд№ҹжҳҜдёҖдёӘи¶…зұ»(еҚізҲ¶зұ»)гҖӮNumberзұ»еұһдәҺjava.langеҢ…пјҢжүҖжңүзҡ„еҢ…иЈ…зұ»пјҲеҰӮDoubleгҖҒFloatгҖҒByteгҖҒShortгҖҒIntegerд»ҘеҸҠLongпјүйғҪжҳҜжҠҪиұЎзұ»Numberзҡ„еӯҗзұ»гҖӮ

Numberзұ»е®ҡд№үдәҶдёҖдәӣжҠҪиұЎж–№жі•пјҢд»Ҙеҗ„з§ҚдёҚеҗҢж•°еӯ—ж јејҸиҝ”еӣһеҜ№иұЎзҡ„еҖјгҖӮеҰӮXxxValue()ж–№жі•пјҢе®ғе°ҶNumberеҜ№иұЎиҪ¬жҚўдёәXxxж•°жҚ®зұ»еһӢзҡ„еҖје№¶иҝ”еӣһгҖӮиҝҷдәӣж–№жі•еҰӮдёӢжүҖзӨәпјҡ

ж–№жі• иҜҙжҳҺ
byteValue() иҝ”еӣһbyteзұ»еһӢзҡ„еҖј
doubleValue() иҝ”еӣһdoubleзұ»еһӢзҡ„еҖј
floatValue() иҝ”еӣһfloatзұ»еһӢзҡ„еҖј
intValue() иҝ”еӣһintзұ»еһӢзҡ„еҖј
longValue() иҝ”еӣһlongзұ»еһӢзҡ„еҖј
shortValue() иҝ”еӣһshortзұ»еһӢзҡ„еҖј

жҠҪиұЎзұ»дёҚиғҪзӣҙжҺҘе®һдҫӢеҢ–пјҢиҖҢжҳҜеҝ…йЎ»е®һдҫӢеҢ–е…¶е…·дҪ“зҡ„еӯҗзұ»гҖӮеҰӮдёӢд»Јз Ғжј”зӨәдәҶNumberзұ»зҡ„дҪҝз”Ёпјҡ

Characterзұ»

Charaterзұ»жҳҜеӯ—з¬Ұж•°жҚ®зұ»еһӢcharзҡ„еҢ…иЈ…зұ»гҖӮCharaterзұ»зҡ„еҜ№иұЎеҢ…еҗ«зұ»еһӢдёәcharзҡ„еҚ•дёӘеӯ—ж®өпјҢиҝҷдёҖиғҪжҠҠеҹәжң¬ж•°жҚ®зұ»еһӢеҪ“еҜ№иұЎжқҘеӨ„зҗҶгҖӮеёёз”Ёж–№жі•еҰӮдёӢпјҡ

ж–№жі• жҸҸиҝ°
Character(value) жһ„йҖ дёҖдёӘж–°еҲҶй…Қзҡ„CharacterеҜ№иұЎпјҢз”Ёд»ҘиЎЁзӨәжҢҮе®ҡзҡ„charеҖјгҖӮ
charValue() иҝ”еӣһжӯӨCharacterеҜ№иұЎзҡ„еҖјпјҢжӯӨеҜ№иұЎиЎЁзӨәеҹәжң¬charеҖј
compareTo(anotherCharacter) ж №жҚ®ж•°еӯ—жҜ”иҫғдёӨдёӘCharacterеҜ№иұЎ
equals(anotherCharacter) е°ҶжӯӨеҜ№иұЎдёҺжҢҮе®ҡеҜ№иұЎжҜ”иҫғпјҢеҪ“дё”д»…еҪ“еҸӮж•°дёҚжҳҜnullпјҢиҖҢжҳҜдёҖдёӘдёҺжӯӨеҜ№иұЎеҢ…еҗ«зӣёеҗҢcharеҖјзҡ„CharacterеҜ№иұЎж—¶пјҢз»“жһңжүҚжҳҜtrue
isDigir(ch) зЎ®е®ҡжҢҮе®ҡеӯ—з¬ҰжҳҜеҗҰдёәж•°еӯ—пјҢеҰӮжһңйҖҡиҝҮCharacter.getType(ch)жҸҗдҫӣзҡ„еӯ—з¬Ұзҡ„еёёи§„зұ»еһӢдёәDECIMAL_DIGIT_NUMBERпјҢеҲҷеӯ—з¬Ұдёәж•°еӯ—
isLetter(codePoint) зЎ®е®ҡжҢҮе®ҡеӯ—з¬ҰпјҲUnicodeд»Јз ҒзӮ№пјүжҳҜеҗҰдёәеӯ—жҜҚ
