еҲӣе»әеӯ—з¬ҰдёІ

еӯ—з¬ҰдёІеҸҳйҮҸеҝ…йЎ»з»ҸиҝҮеҲқе§ӢеҢ–жүҚиғҪдҪҝз”ЁгҖӮ

  • е…ій”®еӯ—е’Ңжһ„йҖ ж–№жі•еҲӣе»ә

    Stringзұ»зҡ„жһ„йҖ ж–№жі•жңүеӨҡз§ҚйҮҚиҪҪдҝ®йҘ°пјҢжҜҸз§Қдҝ®йҘ°йғҪеҸҜд»Ҙе®ҡд№үеӯ—з¬ҰдёІгҖӮ

    1. String()

      еҲқе§ӢеҢ–дёҖдёӘж–°еҲӣе»әзҡ„StringеҜ№иұЎпјҢиЎЁзӨәдёҖдёӘз©әеӯ—з¬ҰеәҸеҲ—гҖӮ

    2. String(String original)

      еҲқе§ӢеҢ–дёҖдёӘж–°еҲӣе»әзҡ„StringеҜ№иұЎпјҢдҪҝе…¶иЎЁзӨәдёҖдёӘдёҺеҸӮж•°зӣёеҗҢзҡ„еӯ—з¬ҰеәҸеҲ—гҖӮжҚўеҸҘиҜқиҜҙпјҢж–°еҲӣе»әзҡ„еӯ—з¬ҰдёІжҳҜиҜҘеҸӮж•°еӯ—з¬ҰдёІзҡ„еүҜжң¬гҖӮ

      иҝҷйҮҢstr1е’Ңstr2еҖјжҳҜзӣёзӯүзҡ„гҖӮ

    3. String(char[]value)

      еҲҶй…ҚдёҖдёӘж–°зҡ„еӯ—з¬ҰдёІпјҢе°ҶеҸӮж•°дёӯзҡ„еӯ—з¬Ұж•°з»„е…ғзҙ е…ЁйғЁеҸҳдёәеӯ—з¬ҰдёІгҖӮиҜҘеӯ—з¬Ұж•°з»„зҡ„еҶ…е®№е·Іиў«еӨҚеҲ¶пјҢеҗҺз»ӯеҜ№еӯ—з¬Ұж•°з»„зҡ„дҝ®ж”№дёҚдјҡеҪұе“Қж–°еҲӣе»әзҡ„еӯ—з¬ҰдёІгҖӮ

      дёҠиҝ°sCharеҸҳйҮҸзҡ„еҖјжҳҜеӯ—з¬ҰдёІHelloгҖӮеҚідҪҝеңЁеҲӣе»әеӯ—з¬ҰдёІд№ӢеҗҺпјҢеҜ№aж•°з»„дёӯзҡ„з¬¬дәҢдёӘе…ғзҙ иҝӣиЎҢдәҶдҝ®ж”№пјҢдҪҶжңӘеҪұе“ҚsCharзҡ„еҖјгҖӮ

    4. String(char[]value,int offset,int count)

      еҲҶй…ҚдёҖдёӘж–°зҡ„StringпјҢе®ғеҢ…еҗ«жқҘиҮӘиҜҘж•°з»„еҸӮж•°дёҖдёӘеӯҗж•°з»„зҡ„еӯ—з¬ҰгҖӮoffsetеҸӮж•°жҳҜеӯҗж•°з»„з¬¬дёҖдёӘеӯ—з¬Ұзҡ„зҙўеј•пјҢcountеҸӮж•°жҢҮе®ҡеӯҗж•°з»„зҡ„й•ҝеәҰгҖӮиҜҘеӯҗж•°з»„зҡ„еҶ…е®№е·Іиў«иөӢеҖјпјҢеҗҺз»ӯеҜ№еӯ—з¬Ұж•°з»„зҡ„дҝ®ж”№дёҚдјҡеҪұе“Қж–°еҲӣе»әзҡ„еӯ—з¬ҰдёІгҖӮ

      дёҠиҝ°sCharеҸҳйҮҸзҡ„еҖјжҳҜеӯ—з¬ҰдёІ"ello"гҖӮеӯ—з¬ҰдёІеҜ№иұЎеҲӣе»әеҗҺпјҢеҚідҪҝеңЁеҗҺйқўдҝ®ж”№дәҶaж•°з»„дёӯзҡ„з¬¬дәҢдёӘе…ғзҙ зҡ„еҖјпјҢеҜ№sCharзҡ„еҖјд№ҹжІЎжңүд»»дҪ•еҪұе“ҚгҖӮ

  • еӯ—з¬Ұж•°з»„еҸӮж•°еҲӣе»ә

жіЁж„Ҹ:String зұ»жҳҜдёҚеҸҜж”№еҸҳзҡ„пјҢжүҖд»ҘдҪ дёҖж—ҰеҲӣе»әдәҶ String еҜ№иұЎпјҢйӮЈе®ғзҡ„еҖје°ұж— жі•ж”№еҸҳдәҶ,еҰӮжһңйңҖиҰҒеҜ№еӯ—з¬ҰдёІеҒҡеҫҲеӨҡдҝ®ж”№пјҢйӮЈд№Ҳеә”иҜҘйҖүжӢ©дҪҝз”Ё StringBuffer & StringBuilder зұ»гҖӮ

StringиҪ¬жҚўдёәint

Stringеӯ—з¬ҰдёІиҪ¬ж•ҙеһӢintжңүд»ҘдёӢдёӨз§Қж–№ејҸпјҡ

  • Integer.paeseInt(str)
  • Integer.valueOf(str).intValue()

жіЁж„ҸпјҡIntegerжҳҜдёҖдёӘзұ»пјҢжҳҜintеҹәжң¬ж•°жҚ®зұ»еһӢзҡ„е°ҒиЈ…зұ»гҖӮ

жіЁж„ҸпјҡStringиҪ¬жҚўintж—¶пјҢStringзҡ„еҖјдёҖе®ҡжҳҜиҜҒд№ҰпјҢеҗҰеҲҷдјҡжҠҘж•°еӯ—иҪ¬жҚўејӮеёё(java.lang.NumberFormatException)гҖӮ

intиҪ¬жҚўдёәString

ж•ҙеҪўintиҪ¬Stringеӯ—з¬ҰдёІзұ»еһӢжңүд»ҘдёӢдёүз§Қж–№жі•пјҡ

  • String s = String.valueOf(i);
  • String s = Integer,.toString(i);
  • String s = "" + i;

жіЁпјҡдҪҝз”Ёз¬¬дёүз§Қж–№жі•зӣёеҜ№з¬¬дёҖз¬¬дәҢдёӯиҖ—ж—¶жҜ”иҫғеӨ§гҖӮеңЁдҪҝз”Ёз¬¬дёҖз§ҚvalueOf()ж–№жі•ж—¶пјҢжіЁж„ҸvalueOf()жӢ¬еҸ·дёӯзҡ„еҖјдёҚиғҪдёәз©әпјҢеҗҰеҲҷдјҡжҠҘз©әжҢҮй’ҲејӮеёё(NullPointerException)гҖӮ

valueOf()гҖҒparse()е’ҢtoString()

  1. valueOf()

    valueOf()ж–№жі•е°Ҷж•°жҚ®зҡ„еҶ…йғЁж јејҸиҪ¬жҚўдёәеҸҜиҜ»зҡ„еҪўејҸгҖӮе®ғжҳҜдёҖз§ҚйқҷжҖҒж–№жі•пјҢеҜ№жүҖжңүJavaеҶ…зҪ®зҡ„зұ»еһӢпјҢеңЁеӯ—з¬ҰдёІеҶ…иў«йҮҚиҪҪпјҢд»ҘдҫҝжҜҸдёҖз§Қзұ»еһӢйғҪиў«иҪ¬жҚўдёәеӯ—з¬ҰдёІгҖӮvalueOf()ж–№жі•иҝҳиў«зұ»еһӢObjectйҮҚиҪҪпјҢжүҖд»ҘеҲӣе»әзҡ„д»»дҪ•еҪўејҸзұ»зҡ„еҜ№иұЎд№ҹеҸҜиў«з”ЁдҪңдёҖдёӘеҸӮж•°пјҢиҝҷжҳҜе®ғзҡ„еҮ з§ҚеҪўејҸпјҡ

    дёҺеүҚйқўи®Ёи®әзҡ„дёҖж ·пјҢи°ғз”ЁvalueOf()ж–№жі•еҸҜд»Ҙеҫ—еҲ°е…¶д»–зұ»еһӢж•°жҚ®зҡ„еӯ—з¬ҰдёІеҪўејҸгҖӮеҜ№еҗ„з§Қж•°жҚ®зұ»еһӢпјҢеҸҜд»ҘзӣҙжҺҘи°ғз”Ёиҝҷз§Қж–№жі•еҫ—еҲ°еҗҲзҗҶзҡ„еӯ—з¬ҰдёІеҪўејҸгҖӮжүҖжңүзҡ„з®ҖеҚ•зұ»еһӢж•°жҚ®иҪ¬жҚўжҲҗзӣёеә”дәҺе®ғд»¬зҡ„жҷ®йҖҡеӯ—з¬ҰдёІеҪўејҸгҖӮд»»дҪ•дј йҖ’з»ҷvalueOf()ж–№жі•зҡ„еҜ№иұЎйғҪе°Ҷиҝ”еӣһеҜ№иұЎзҡ„toString()ж–№жі•и°ғз”Ёзҡ„з»“жһңгҖӮдәӢе®һдёҠпјҢд№ҹеҸҜд»ҘйҖҡиҝҮзӣҙжҺҘи°ғз”Ё toString()ж–№жі•иҖҢеҫ—еҲ°зӣёеҗҢзҡ„з»“жһңгҖӮ

    еҜ№дәҺеӨ§еӨҡж•°ж•°з»„пјҢvalueOf()ж–№жі•иҝ”еӣһдёҖдёӘз®ҖеҚ•жҷҰж¶©зҡ„еӯ—з¬ҰдёІпјҢиҝҷиҜҙжҳҺе®ғжҳҜдёҖдёӘжҹҗз§Қзұ»еһӢзҡ„ж•°з»„гҖӮз„¶иҖҢеҜ№дәҺеӯ—з¬Ұж•°з»„пјҢе®ғеҲӣе»әдёҖдёӘеҢ…еҗ«дәҶеӯ—з¬Ұж•°з»„дёӯзҡ„еӯ—з¬Ұзҡ„еӯ—з¬ҰдёІеҜ№иұЎгҖӮvalueOf()ж–№жі•жңүдёҖз§Қзү№е®ҡеҪўејҸе…Ғи®ёжҢҮе®ҡеӯ—з¬Ұж•°з»„зҡ„дёҖдёӘеӯҗйӣҶгҖӮе®ғе…·жңүеҰӮдёӢзҡ„дёҖиҲ¬еҪўејҸпјҡ

    иҝҷйҮҢcharsжҳҜеӯҳж”ҫеӯ—з¬Ұзҡ„ж•°з»„пјҢstartIndexжҳҜеӯ—з¬Ұж•°з»„дёӯжңҹжңӣеҫ—еҲ°зҡ„еӯҗеӯ—з¬ҰдёІзҡ„йҰ–еӯ—з¬ҰдёӢж ҮпјҢnumCharsжҢҮе®ҡеӯҗеӯ—з¬ҰдёІзҡ„й•ҝеәҰгҖӮ

  2. parse()

    parseXxx(String)иҝҷз§ҚеҪўејҸпјҢжҳҜжҢҮжҠҠеӯ—з¬ҰдёІиҪ¬жҚўдёәXxxзұ»еһӢеҖјпјҢе…¶дёӯXxxеҜ№еә”дёҚеҗҢзҡ„ж•°жҚ®зұ»еһӢпјҢеҰӮintеһӢе’ҢfloatеһӢгҖӮ

  3. toString()

    toString()еҸҜд»ҘжҠҠдёҖдёӘеј•з”Ёзұ»еһӢиҪ¬жҚўдёәStringеӯ—з¬ҰдёІзұ»еһӢгҖӮ

еӯ—з¬ҰдёІжӢјжҺҘ

Stringеӯ—з¬ҰдёІжҳҜдёҚеҸҜеҸҳеӯ—з¬ҰдёІпјҢдҪҶеҸҜд»ҘиҝӣиЎҢжӢјжҺҘпјҢдјҡдә§з”ҹдёҖдёӘж–°зҡ„еҜ№иұЎгҖӮ

  1. Stringзұ»жҸҗдҫӣдәҶиҝһжҺҘдёӨдёӘеӯ—з¬ҰдёІзҡ„ж–№жі•пјҡstring1.concat(string2);иҝ”еӣһ string2 иҝһжҺҘ string1 зҡ„ж–°еӯ—з¬ҰдёІгҖӮд№ҹеҸҜд»ҘеҜ№еӯ—з¬ҰдёІеёёйҮҸдҪҝз”Ё concat() ж–№жі•пјҢеҰӮпјҡ"жҲ‘зҡ„еҗҚеӯ—жҳҜ".concat("hugh");

  2. дҪҝз”Ё+ж“ҚдҪңз¬ҰжқҘиҝһжҺҘеӯ—з¬ҰдёІпјҡ"Hello, " + "hugh" + "!";

    дҪҶжҳҜеңЁеҫӘзҺҜдёӯпјҢзӣҙжҺҘдҪҝз”Ё+иҝӣиЎҢеӯ—з¬ҰдёІжӢјжҺҘжҜҸж¬ЎеҫӘзҺҜйғҪдјҡеҲӣе»әж–°зҡ„еӯ—з¬ҰдёІеҜ№иұЎпјҢз„¶еҗҺдёўжҺүж—§зҡ„еӯ—з¬ҰдёІпјҢиҝҷж ·з»қеӨ§йғЁеҲҶеӯ—з¬ҰдёІйғҪжҳҜдёҙж—¶еҜ№иұЎпјҢдёҚдҪҶжөӘиҙ№еҶ…еӯҳиҝҳдјҡеҪұе“Қ GC ж•ҲзҺҮгҖӮ

    ж ҮеҮҶеә“дёӯзҡ„StringBuilderпјҢжҳҜдёҖдёӘеҸҜеҸҳеҜ№иұЎпјҢеҸҜд»Ҙйў„еҲҶй…Қзј“еҶІеҢәпјҢиҝҷж ·еңЁе…¶дёӯж–°еўһеӯ—з¬Ұж—¶пјҢе°ұдёҚдјҡеҲӣе»әж–°зҡ„дёҙж—¶еҜ№иұЎгҖӮ

    StringBuilderиҝҳеҸҜд»ҘиҝӣиЎҢй“ҫејҸж“ҚдҪңпјҡ

    жҹҘзңӢStringBuilderзҡ„жәҗз ҒпјҢеҸҜд»ҘеҸ‘зҺ°пјҡиҝӣиЎҢй“ҫејҸж“ҚдҪңзҡ„е…ій”®жҳҜпјҢе®ҡд№үзҡ„append()ж–№жі•дјҡиҝ”еӣһthisпјҢиҝҷж ·пјҢе°ұеҸҜд»ҘдёҚж–ӯи°ғз”ЁиҮӘиә«зҡ„е…¶д»–ж–№жі•гҖӮStringBuilderеҸҜд»Ҙж”ҜжҢҒй“ҫејҸж“ҚдҪңпјҢе®һзҺ°й“ҫејҸж“ҚдҪңзҡ„е…ій”®жҳҜиҝ”еӣһе®һдҫӢжң¬иә«гҖӮ

    д»ҝз…§StringBuilderпјҢеҸҜд»Ҙи®ҫи®Ўж”ҜжҢҒй“ҫејҸж“ҚдҪңзҡ„зұ»гҖӮдҫӢеҰӮпјҢдёҖдёӘеҸҜд»ҘдёҚж–ӯеўһеҠ зҡ„и®Ўж•°еҷЁпјҡ

    еҜ№дәҺжҷ®йҖҡзҡ„еӯ—з¬ҰдёІ+ж“ҚдҪңпјҢдёҚйңҖиҰҒе°Ҷе…¶ж”№еҶҷдёәStringBuilderпјҢеӣ дёәJavaзј–иҜ‘еҷЁеңЁзј–иҜ‘ж—¶е°ұдјҡиҮӘеҠЁең°жҠҠеӨҡдёӘиҝһз»ӯзҡ„+ж“ҚдҪңзј–з ҒдёәStringConcatFactoryзҡ„ж“ҚдҪңгҖӮеңЁиҝҗиЎҢжңҹй—ҙпјҢStringConcatFactoryдјҡе°Ҷеӯ—з¬ҰдёІиҝһжҺҘж“ҚдҪңдјҳеҢ–дёәж•°з»„еӨҚеҲ¶жҲ–иҖ…StringBuilderж“ҚдҪңгҖӮ

  1. з”ЁStringJoinerжқҘеӨ„зҗҶз”ЁеҲҶйҡ”з¬ҰжӢјжҺҘж•°з»„зҡ„йңҖжұӮгҖӮ

    жҹҘзңӢStringJoinerзҡ„жәҗз ҒеҸҜд»ҘеҸ‘зҺ°пјҡStringJoinerеҶ…йғЁе®һйҷ…дёҠе°ұжҳҜдҪҝз”ЁдәҶStringBuilderпјҢжүҖд»ҘжӢјжҺҘж•ҲзҺҮе’ҢStringBuilderеҮ д№ҺдёҖжЁЎдёҖж ·гҖӮ

    StringиҝҳжҸҗдҫӣдәҶдёҖдёӘйқҷжҖҒж–№жі•join()пјҢиҝҷдёӘж–№жі•еңЁеҶ…йғЁдҪҝз”ЁдәҶStringJoinerжқҘжӢјжҺҘеӯ—з¬ҰдёІпјҢеңЁдёҚйңҖиҰҒжҢҮе®ҡвҖқејҖеӨҙвҖқе’ҢвҖқз»“е°ҫвҖқзҡ„ж—¶еҖҷпјҢз”ЁString.join()жӣҙж–№дҫҝ

еӯ—з¬ҰдёІй•ҝеәҰ

з”ЁдәҺиҺ·еҸ–жңүе…іеҜ№иұЎзҡ„дҝЎжҒҜзҡ„ж–№жі•з§°дёәи®ҝй—®еҷЁж–№жі•гҖӮStringзұ»зҡ„дёҖдёӘи®ҝй—®еҷЁж–№жі•жҳҜlength()ж–№жі•пјҢе®ғиҝ”еӣһеӯ—з¬ҰдёІеҜ№иұЎеҢ…еҗ«зҡ„еӯ—з¬Ұж•°гҖӮиҜӯжі•еҪўејҸеҰӮдёӢпјҡ

еӯ—з¬ҰдёІеӨ§е°ҸеҶҷиҪ¬жҚў

StringBufferзұ»зҡ„setCharAt()ж–№жі•з”ЁдәҺеңЁеӯ—з¬ҰдёІзҡ„жҢҮе®ҡзҙўеј•дҪҚзҪ®жӣҝжҚўдёҖдёӘеӯ—з¬ҰгҖӮиҜӯжі•еҰӮдёӢпјҡ

иҜҘж–№жі•зҡ„дҪңз”ЁжҳҜдҝ®ж”№еҜ№иұЎдёӯзҙўеј•еҖјдёәindexдҪҚзҪ®зҡ„еӯ—з¬Ұдёәж–°зҡ„еӯ—з¬ҰchгҖӮ

еҸҚиҪ¬еӯ—з¬ҰдёІ

StringBufferзұ»дёӯзҡ„reverse()ж–№жі•з”ЁдәҺе°Ҷеӯ—з¬ҰдёІеәҸеҲ—з”Ёе…¶еҸҚиҪ¬зҡ„еҪўејҸеҸ–д»ЈгҖӮиҜӯжі•ж јејҸеҰӮдёӢпјҡ

еҲ йҷӨеӯ—з¬ҰдёІ

  1. deleteCharAt()ж–№жі•

    deleteCharAt()ж–№жі•з”ЁдәҺз§»йҷӨеәҸеҲ—дёӯжҢҮе®ҡдҪҚзҪ®зҡ„еӯ—з¬ҰгҖӮиҜӯжі•еҰӮдёӢпјҡ

    deleteCharAt()ж–№жі•зҡ„дҪңз”ЁжҳҜеҲ йҷӨжҢҮе®ҡдҪҚзҪ®зҡ„еӯ—з¬ҰпјҢз„¶еҗҺе°Ҷеү©дҪҷзҡ„еҶ…е®№еҪўжҲҗдёҖдёӘж–°зҡ„еӯ—з¬ҰдёІгҖӮ

  2. delete()ж–№жі•

    delete()ж–№жі•з”ЁдәҺз§»йҷӨеәҸеҲ—дёӯеӯ—з¬ҰдёІзҡ„еӯ—з¬ҰпјҢиҜӯжі•еҰӮдёӢпјҡ

StringгҖҒStringBufferгҖҒStringBuilderзұ»зҡ„еҢәеҲ«

еңЁJavaдёӯеӯ—з¬ҰдёІеұһдәҺеҜ№иұЎпјҢJavaжҸҗдҫӣдәҶStringзұ»жқҘеҲӣе»әе’Ңж“ҚдҪңеӯ—з¬ҰдёІгҖӮStringзұ»жҳҜдёҚеҸҜеҸҳзұ»пјҢеҚідёҖж—ҰдёҖдёӘStringеҜ№иұЎиў«еҲӣе»әд»ҘеҗҺпјҢеҢ…еҗ«еңЁиҝҷдёӘеҜ№иұЎдёӯзҡ„еӯ—з¬ҰеәҸеҲ—жҳҜдёҚеҸҜж”№еҸҳзҡ„пјҢзӣҙиҮіиҝҷдёӘеҜ№иұЎиў«й”ҖжҜҒгҖӮ

Java жҸҗдҫӣдәҶдёӨдёӘеҸҜеҸҳеӯ—з¬ҰдёІзұ»StringBufferе’ҢStringBuilderпјҢдёӯж–Үзҝ»иҜ‘дёәвҖңеӯ—з¬ҰдёІзј“еҶІеҢәвҖқгҖӮ

StringBuilderзұ»жҳҜ JDK 1.5 ж–°еўһзҡ„зұ»пјҢе®ғд№ҹд»ЈиЎЁеҸҜеҸҳеӯ—з¬ҰдёІеҜ№иұЎгҖӮе®һйҷ…дёҠпјҢStringBuilderе’Ң StringBufferеҠҹиғҪеҹәжң¬зӣёдјјпјҢж–№жі•д№ҹе·®дёҚеӨҡгҖӮдёҚеҗҢзҡ„жҳҜпјҢStringBufferжҳҜзәҝзЁӢе®үе…Ёзҡ„пјҢиҖҢ StringBuilderеҲҷжІЎжңүе®һзҺ°зәҝзЁӢе®үе…ЁеҠҹиғҪпјҢжүҖд»ҘжҖ§иғҪз•Ҙй«ҳгҖӮеӣ жӯӨеңЁйҖҡеёёжғ…еҶөдёӢпјҢеҰӮжһңйңҖиҰҒеҲӣе»әдёҖдёӘеҶ…е®№еҸҜеҸҳзҡ„еӯ—з¬ҰдёІеҜ№иұЎпјҢеҲҷеә”иҜҘдјҳе…ҲиҖғиҷ‘дҪҝз”ЁStringBuilderзұ»гҖӮ

StringBufferгҖҒStringBuilderгҖҒStringдёӯйғҪе®һзҺ°дәҶCharSequenceжҺҘеҸЈгҖӮCharSequenceжҳҜдёҖдёӘе®ҡд№үеӯ—з¬ҰдёІж“ҚдҪңзҡ„жҺҘеҸЈпјҢе®ғеҸӘеҢ…жӢ¬length()гҖҒcharAt(int index)гҖҒsubSequence(int start, int end)иҝҷеҮ дёӘ APIгҖӮ

StringBufferгҖҒStringBuilderгҖҒStringеҜ№CharSequenceжҺҘеҸЈзҡ„е®һзҺ°иҝҮзЁӢдёҚдёҖж ·пјҢеҰӮдёӢеӣҫ 1 жүҖзӨәпјҡ

еҸҜи§ҒпјҢStringзӣҙжҺҘе®һзҺ°дәҶCharSequenceжҺҘеҸЈпјҢStringBuilderе’ҢStringBufferйғҪжҳҜеҸҜеҸҳзҡ„еӯ—з¬ҰеәҸеҲ—пјҢе®ғд»¬йғҪз»§жүҝдәҺAbstractStringBuilderпјҢе®һзҺ°дәҶCharSequenceжҺҘеҸЈгҖӮ

жҖ»з»“

StringжҳҜ Java дёӯеҹәзЎҖдё”йҮҚиҰҒзҡ„зұ»пјҢиў«еЈ°жҳҺдёәfinal classпјҢжҳҜдёҚеҸҜеҸҳеӯ—з¬ҰдёІгҖӮеӣ дёәе®ғзҡ„дёҚеҸҜеҸҳжҖ§пјҢжүҖд»ҘжӢјжҺҘеӯ—з¬ҰдёІж—¶еҖҷдјҡдә§з”ҹеҫҲеӨҡж— з”Ёзҡ„дёӯй—ҙеҜ№иұЎпјҢеҰӮжһңйў‘з№Ғзҡ„иҝӣиЎҢиҝҷж ·зҡ„ж“ҚдҪңеҜ№жҖ§иғҪжңүжүҖеҪұе“ҚгҖӮ

StringBufferе°ұжҳҜдёәдәҶи§ЈеҶіеӨ§йҮҸжӢјжҺҘеӯ—з¬ҰдёІж—¶дә§з”ҹеҫҲеӨҡдёӯй—ҙеҜ№иұЎй—®йўҳиҖҢжҸҗдҫӣзҡ„дёҖдёӘзұ»гҖӮе®ғжҸҗдҫӣдәҶ appendе’Ңaddж–№жі•пјҢеҸҜд»Ҙе°Ҷеӯ—з¬ҰдёІж·»еҠ еҲ°е·ІжңүеәҸеҲ—зҡ„жң«е°ҫжҲ–жҢҮе®ҡдҪҚзҪ®пјҢе®ғзҡ„жң¬иҙЁжҳҜдёҖдёӘзәҝзЁӢе®үе…Ёзҡ„еҸҜдҝ®ж”№зҡ„еӯ—з¬ҰеәҸеҲ—гҖӮеңЁеҫҲеӨҡжғ…еҶөдёӢжҲ‘д»¬зҡ„еӯ—з¬ҰдёІжӢјжҺҘж“ҚдҪңдёҚйңҖиҰҒзәҝзЁӢе®үе…ЁпјҢжүҖд»Ҙ StringBuilderзҷ»еңәдәҶгҖӮStringBuilderжҳҜ JDK1.5 еҸ‘еёғзҡ„пјҢе®ғе’ҢStringBufferжң¬иҙЁдёҠжІЎд»Җд№ҲеҢәеҲ«пјҢе°ұжҳҜеҺ»жҺүдәҶдҝқиҜҒзәҝзЁӢе®үе…Ёзҡ„йӮЈйғЁеҲҶпјҢеҮҸе°‘дәҶејҖй”ҖгҖӮ
